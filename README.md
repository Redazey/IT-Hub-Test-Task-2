# IT-Hub-Test-Task-2
.NET C# Backend в качестве тестового задания в компанию IT Hub «Северстали»

# API
Реализованное API соответствует всем предьявленным в задании требованиям

Из бонусных заданий не выполненно только №2

Структура проекта реализована в соответствии с принципами SOLID (Идеи для улучшения №1)

# Tests
Тесты полностью покрывают все сервисы API (в том числе все поля для фильтрации рулонов)

Реализованы через InMemoryDB, т.е. от реальной БД не зависят
